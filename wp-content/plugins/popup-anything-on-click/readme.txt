== Installation ==

1. Upload the 'popup-anything-on-click' folder to the '/wp-content/plugins/' directory.
2. Activate the "popup-anything-on-click" list plugin through the 'Plugins' menu in WordPress.
3. Check the Popup Anything Menu button and start adding the popup.

== Frequently Asked Questions ==

= I want to change button design as per my theme =

You can use <code>.paoc-popup-btn</code> class to change the button design as per your theme. Add the bellow custom CSS under custom CSS section of your theme and change the color and other values as per your need. e.g:

<code>
.paoc-popup-btn {
	padding: 10px 20px;
	background: #e91e63;
	border-radius: 5px;
	color: #fff;
	text-decoration: none !important;
}
</code>
